char*() rtc_get_current_time ( )

Returns the current time string in the format hh:mm:ss, size given by TIME_STRING_SIZE.

Returns
pointer to the current time string. This string should not be freed. Returns null in case the time has not been initialized yet, so it should be tested if the value is NULL before using it.
